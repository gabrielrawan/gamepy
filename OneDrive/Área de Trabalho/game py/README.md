# 🎮 Recruiter Game

Um jogo Python interativo sobre contratação de desenvolvedores. Escolha o melhor candidato!

## 📋 Descrição

Este é um jogo de terminal educativo onde você assume o papel de um recrutador e precisa escolher entre 3 candidatos:
- **A** - Fala bem, mas sem projetos reais
- **B** - Conhece a teoria, mas nunca entregou nada
- **C** - Constrói, entrega e melhora constantemente

## 🚀 Como Rodar

### Localmente (seu computador):
```bash
git clone https://github.com/gabrielrawan/gamepy.git
cd gamepy
python game.py
```

### Online (sem instalar nada):
1. Abra o repositório no GitHub
2. Clique em **Code** → **Codespaces** → **Create codespace on master**
3. No terminal, digite: `python game.py`

## 🎯 Como Jogar

1. O jogo mostra 3 candidatos
2. Digite **A**, **B**, ou **C** para escolher
3. Veja o resultado da sua escolha
4. Digite **yes** para jogar novamente ou **no** para sair

## 💻 Tecnologias

- **Python 3.x**
- **Terminal/CLI**
